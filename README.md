## Project Description
This project was built as part of Web Foundations training to practice core HTML concepts including semantic structure, multi-page navigation, accessibility basics, and GitHub workflow. The project includes multiple HTML pages (Home, About, Projects, Contact) created to strengthen understanding of page structure and layout.

## Technologies Used
- HTML5
- CSS
- Git & GitHub

## Features
- Multi-page navigation system
- Contact form
- Accessibility improvements 
- Semantic HTML structure

## How to Run
1. Clone this repository: git clone https://github.com/Mshi-dev15/iyf-s10-week-01-Mshi-dev15.git
2. Open the project folder
3. Double click 'index.html'

## Lessons Learned
- How to structure websites using semantic HTML  
- Importance of accessibility in web development and how to run Lighthouse audits  
- How to create functional navigation with highlighted current pages  
- Proper use of forms, labels, and placeholder content  
- Git workflow: staging, committing, and pushing files to GitHub  
- Organizing a project folder with HTML, images, and documentation

## Challenges Faced
- Figuring out how to properly link multiple HTML pages together  
- Ensuring images display correctly and paths are accurate  
- Highlighting the current page in the navigation menu  
- Completing the accessibility audit on a local file since Lighthouse prefers HTTPS  
- Organizing screenshots and files in a clean folder structure for GitHub
